# プロダクト概要

Crypto Watchは、スマートウォッチクライアント向けに最適化された暗号通貨価格監視アプリケーションです。システムは以下で構成されています：

- **Flutterモバイルアプリ**: リアルタイムの暗号通貨価格を表示するクロスプラットフォームモバイルアプリケーション（iOS、Android、Web、デスクトップ）
- **AWSサーバーレスバックエンド**: 5分ごとに自動更新される暗号通貨価格データを提供するRESTful API

## 主な機能

- 20種類以上の主要暗号通貨（BTC、ETH、ADAなど）のリアルタイム価格追跡
- スマートウォッチ最適化されたAPIレスポンス（最小ペイロード、gzip圧縮）
- APIキー認証とレート制限（キーあたり100リクエスト/分）
- 外部データソースからの5分ごとの自動価格更新
- マルチ環境デプロイ（dev、staging、production）
- 監視用のヘルスチェックエンドポイント

## 対象プラットフォーム

- iOSおよびAndroidモバイルデバイス
- Webブラウザ
- デスクトップ（Windows、macOS、Linux）
- スマートウォッチクライアント（主要な最適化対象）
